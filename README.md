# PG Intelligence Website

Site institucional da **PG Intelligence**, focado em apresentar serviços de IA, automação e desenvolvimento web com uma identidade visual moderna e abordagem orientada a resultados.

## O que este site faz

O projeto foi construído para:

- apresentar os principais serviços da PG Intelligence;
- mostrar diferenciais e provas de entrega;
- destacar os fundadores e a proposta da empresa;
- facilitar contato comercial via canais diretos.

## Principais seções

- **Hero:** posicionamento da marca e chamadas para ação.
- **Serviços (cards interativos):** cards com efeito de flip no clique, exibindo detalhes no verso.
- **Por que escolher a PG:** seção de autoridade com diferenciais da empresa.
- **Provas:** resultados e depoimentos.
- **Sobre os criadores:** seção com carrossel (logo + cards dos fundadores).
- **Processo:** etapas de trabalho da descoberta ao lançamento.
- **Contato:** bloco final para conversão.

## Stack técnica

- **React + TypeScript**
- **Vite**
- **Tailwind CSS**
- **Lucide React (ícones)**

## Como rodar localmente

1. Instale dependências:

```bash
pnpm install
```

2. Inicie o ambiente de desenvolvimento:

```bash
pnpm dev
```

3. Build de produção:

```bash
pnpm build
```

## Estrutura relevante

- `src/pages/Index.tsx`: página principal com as seções do site.
- `src/components/`: componentes reutilizáveis.
- `src/lib/language.ts`: controle de idioma.
- `index.html`: metadados globais e título da aba.

## Objetivo do projeto

Transformar o site da PG Intelligence em uma vitrine clara, confiável e com boa experiência de navegação, reforçando autoridade e facilitando geração de oportunidades comerciais.
